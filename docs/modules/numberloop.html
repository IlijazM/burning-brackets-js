numberFor

  • numberFor(fun: CallbackFunction, steps?: number): void
    • Defined in src/numberLoop.ts:32

    Will iterate from 0 to the target number and call the callback function with the index for each iteration.

    Call this function from the Number protoype function `Number.prototype.for` like so: 

    (10).for((i) => console.log(i));

    Or alternatively: 

    10..for((i) => console.log(i));

    Parameters

    • fun: CallbackFunction

      the callback function that contains an index as the only parameter.

    • steps: number = 1

      step size for each iteration.

    Returns void
