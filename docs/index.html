typeOf

typeOf is an extension for the already existing typeof but with more distinctions between types. See the differences here:

typeof null; // 'object'.
typeOf(null); // 'null'

class MyClass {}

typeof MyClass; // 'function'
typeOf(MyClass); // 'class'

typeof []; // 'object'
typeOf([]); // 'array'
